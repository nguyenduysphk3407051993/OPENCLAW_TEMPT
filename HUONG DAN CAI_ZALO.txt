🔥 HƯỚNG DẪN CÀI OPENZALO - BIẾN ZALO THÀNH KÊNH CHAT CHO OPENCLAW AI HOÀN TOÀN MIỄN PHÍ
Mình vừa setup xong OpenZalo cho con bot OpenClaw, chạy song song với Telegram luôn. Chia sẻ lại cách cài đúng cho ae khỏi đạp mìn như mình 😅
Xin trân trọng cảm ơn anh Tuyển Hồ đã tạo ra plugin open-source cho ae
📌 OpenZalo là gì?
Plugin cho OpenClaw để dùng tài khoản Zalo cá nhân làm kênh chat với AI. Bot sẽ nhận/gửi tin nhắn, ảnh, voice, file qua Zalo luôn.
Repo: https://github.com/darkamenosa/openzalo
⚡ Yêu cầu
• OpenClaw đã cài và chạy được
• Node.js v22+
• npm
🛠 CÁC BƯỚC CÀI ĐẶT
BƯỚC 1: Cài openzca (CLI tool cho Zalo)
👉 npm i -g openzca
Kiểm tra: openzca --version
BƯỚC 2: Cài plugin OpenZalo vào OpenClaw
⚠️ QUAN TRỌNG: KHÔNG dùng "npm i -g @openclaw/openzalo" — phải cài qua OpenClaw CLI.
👉 cd /tmp
👉 git clone https://github.com/darkamenosa/openzalo.git
👉 openclaw plugins install /tmp/openzalo
BƯỚC 3: Restart Gateway
👉 openclaw gateway restart
Kiểm tra: openclaw plugins list
Phải thấy openzalo ở trạng thái "loaded", không có errors.
BƯỚC 4: Login Zalo bằng QR code
👉 openclaw channels login --channel openzalo
QR code sẽ được lưu ra file. Mở Zalo trên điện thoại → Quét mã QR để đăng nhập.
💡 QR có thời hạn, quét nhanh kẻo hết hạn phải chạy lại lệnh.
BƯỚC 5: Thêm channel config
Sửa file ~/.openclaw/openclaw.json, thêm vào phần "channels":
"openzalo": {
"enabled": true,
"dmPolicy": "pairing",
"groupPolicy": "allowlist",
"groupRequireMention": true,
"sendFailureNotice": true,
"textChunkLimit": 2000
}
Rồi restart: openclaw gateway restart
BƯỚC 6: Pairing (ghép nối)
Lần đầu nhắn tin cho bot trên Zalo, nó sẽ trả về mã pairing.
Approve bằng lệnh:
👉 openclaw pairing approve openzalo XXXXXXXX
Xong! Nhắn lại trên Zalo là bot trả lời 🎉
⚠️ NHỮNG LỖI HAY GẶP
1️⃣ "unknown channel id: openzalo"
→ Plugin chưa được load. Phải cài đúng bằng "openclaw plugins install", KHÔNG phải "npm i -g".
2️⃣ "plugin not found: openzalo"
→ Plugin phải nằm trong ~/.openclaw/extensions/openzalo/ và có file openclaw.plugin.json.
3️⃣ Config invalid sau khi sửa
→ KHÔNG thêm channels.openzalo hay plugins.entries.openzalo TRƯỚC khi plugin được load. Thứ tự đúng: cài plugin → restart → rồi mới thêm channel config.
4️⃣ QR hết hạn
→ Chạy lại openclaw channels login --channel openzalo
5️⃣ Gateway chết sau khi sửa config
→ File backup ở ~/.openclaw/openclaw.json.bak, restore lại rồi restart.
✅ KHẢ NĂNG HỖ TRỢ
Gửi đi: Text ✅ | Ảnh ✅ | Voice ✅ | File ✅ | Video ✅
Nhận vào: Text ✅ | Ảnh ✅ | Voice ✅ (cần whisper) | File ✅ | Video ✅
💡 TIPS
• Chạy song song nhiều channel (Telegram + Zalo + Discord...)
• Dùng "openclaw doctor --non-interactive" để kiểm tra tổng thể
• Zalo giới hạn 2000 ký tự/tin nhắn, plugin tự chia nhỏ
